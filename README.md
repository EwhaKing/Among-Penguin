## 게임 개요
- 제목: 어몽펭귄(Among-Penguin)
- 한 줄 요약: 남극을 촬영하기 위해 투입된 스파이 로봇 펭귄이 되어, 정체를 들키지 않고 펭귄 사회에 스며들어 임무를 완수하라! 
- 장르: 펭귄 시뮬레이션 +  잠입 액션
- 플랫폼: PC
- 개발 엔진: Unity 2D
- 게임 목표: 남극의 다양한 사진 3개 수집
